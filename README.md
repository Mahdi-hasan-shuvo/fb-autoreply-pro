## 🗂️ 3 Powerful Reply Modes

<div align="center">

### MODE 1 — `"all"` — Blanket Reply
```
Every single commenter gets your default reply.
Perfect for: Giveaways, announcements, product launches.

  Anyone comments  ──►  ✅ Default reply sent
```

### MODE 2 — `"specific"` — Smart Reply
```
Checks keywords. Sends the RIGHT reply for the RIGHT question.
Perfect for: E-commerce, service pages, product pages.

  Comment with keyword  ──►  ✅ Custom reply (price/shipping/etc.)
  Comment without match ──►  ✅ Default reply
```

### MODE 3 — `"both"` — Maximum Engagement 🏆 Most Popular
```
Sends default reply to EVERYONE, THEN a custom reply if keyword matched.
2 touchpoints = higher DM rate = more sales.
Perfect for: Aggressive marketing, high-volume ad campaigns.

  Anyone comments          ──►  ✅ Default reply (always)
  Keyword match found      ──►  ⏳ Wait 3 seconds...
                           ──►  ✅ Specific reply (bonus message!)
```
